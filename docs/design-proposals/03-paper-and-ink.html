Element Gallery

The six building blocks of every Event Model slice.

COMMAND

Command

An intent to change state. Represents a user or system action that produces one or more events when accepted.

EVENT

Event

An immutable fact that something happened. The single source of truth in an event-sourced system.

VIEW

View / Read Model

A projection of events into a shape optimized for querying. Drives the UI and informs decisions.

TRIGGER

Trigger

The user-facing entry point -- a form, button, or interaction that initiates a command.

EXCEPTION

Exception

A rejection or failure outcome. Represents business rule violations or error conditions.

AUTOMATION

Automation

A process manager or policy that reacts to events and issues further commands without human input.
