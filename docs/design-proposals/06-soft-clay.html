Element Gallery

C
Command
An intention to change state. Represents a user or system action that may produce events.
E
Event
An immutable fact that something happened. The single source of truth in the model.
V
View
A read model or projection derived from events. What users and systems see and query.
T
Trigger
A user action or external input that initiates a flow. The starting point of a slice.
!
Exception
An error or failure scenario. Captures what happens when a command cannot be fulfilled.
A
Automation
A system-initiated reaction to an event. Triggers downstream commands without user input.
