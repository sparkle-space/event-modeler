Element Gallery

Trigger
User action or external stimulus that initiates a flow. Rendered as a gray rectangle with thick black border.
Command
An intent to change the system state. Bauhaus blue flat fill communicates authority and action.
Event
An immutable fact that has occurred. Bauhaus red signals significance and finality.
View
A read model or projection displayed to users. Bauhaus yellow/gold conveys output and visibility.
Exception
An error or failure state in the flow. Deep red-orange differentiates from standard events.
Automation
A system process reacting to events automatically. Deep blue-violet conveys machine-driven logic.
