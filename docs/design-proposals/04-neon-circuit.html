// Element Gallery

CMD
Command
An intent to change the system state. Validated against business rules before producing events.
EVT
Event
An immutable fact that something happened. The single source of truth in the system.
VW
View
A read-model projection built from events. Optimized for queries and display.
TRG
Trigger
A user action or external input that initiates a command. The entry point of a slice.
EXC
Exception
A failure or rejection path. Captures what happens when a command cannot be fulfilled.
AUT
Automation
A system reaction to an event. Automatically triggers a downstream command without user input.
