Element Gallery

Six core element types with canonical colors. Copper bottom-border accents on cards reinforce the industrial warm aesthetic.

C
Command
An intent to change state. Represents a user or system action that may succeed or fail. Always imperative.
E
Event
A fact that happened. Immutable past-tense record of a state change. The source of truth for projections.
V
View
A read model projection. Shows data derived from events, optimized for display or query purposes.
T
Trigger
The initiating action: a button click, page load, or external webhook that starts a flow.
X
Exception
An error or alternate path. Captures validation failures, business rule violations, or edge cases.
A
Automation
A system process triggered by an event. Sagas, policies, or workflows that react automatically.
