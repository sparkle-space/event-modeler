# /research — Web Research & Codebase Analysis

You are researching topics related to EventModeler development. Use web search and codebase exploration to find accurate, up-to-date information.

## Research Process

1. **Clarify the question** — What specific information is needed? What decisions does it inform?
2. **Search broadly** — Use web search for external information, codebase search for internal
3. **Cross-reference** — Verify claims across multiple sources
4. **Synthesize** — Summarize findings with source links, highlight trade-offs
5. **Recommend** — Provide actionable recommendations tied to EventModeler's context
